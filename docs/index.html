SRM Dispatch

SRM Concrete North Alabama ‚ÄĒ Daily dispatch tool for 18 dump truck drivers. Local-first React PWA. No cloud. No login. One tap, paste, done.

View on GitHub Treasures Hub
React 18 Vite 5 Static PWA No Backend

What It Does

Daily Route Generation

Per-driver routes based on crew, day type, and plant availability. One tap copies the full route to paste into a text message.

Bridgeport Rotation

3-group (A/B/C) continuous weekday cycle, auto-calculated. Never resets ‚ÄĒ holidays skipped, not restarted. 4-week calendar view.

Audible System

Mark any plant DOWN, routes auto-update with substitutions from the pre-defined plant substitution map. Red banner warns dispatchers instantly.

Day Type Toggle

One-click switch between Mon/Wed/Thu and Tue/Fri schedules. Auto-detects on load, override manually.

Special Modes

MH Day, 519 Swap, Curtis Office, Alexis Short Day ‚ÄĒ built-in special schedule modes for real operational scenarios.

Crew Tabs

ALL, DUMP, BRIDGEPORT, 507, 519, 506 ‚ÄĒ filter drivers by crew or plant assignment instantly.

Tech Stack

React 18 Vite 5 Inline Styles (intentional) useState only No backend No database No login Static PWA

Quick Start

git clone https://github.com/thebardchat/srm-dispatch.git
cd srm-dispatch
npm install
npm run dev
# Opens at http://localhost:5173
